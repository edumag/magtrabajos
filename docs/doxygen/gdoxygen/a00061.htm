Descripción detallada

magbackup es un script para generar copias de seguridad.

Con los ficheros creamos una copia por mes y a partir de esta vamos creando archivos por cada día de la semana con los cambios desde la copia por mes.

Para generar las copias se debe crear un fichero de perfil en la carpeta ~/.config/magscripts/magbackup/ficheros/perfil el nombre de la copia que se genera se basa en el nombre del archivo de perfil su formato es:

CS_DIR="/home/eduardo/" CS_COPIAR="Scripts .bashrc"

CS_DIR: Nos indica la carpeta donde situarnos CS_COPIAR: Las subcarpetas o archivos que deseamos copiar

En caso de una copia de mysql en CS_DIR debera añadirse mysql <user> <password> y en CS_COPIAR los nombres de las bases de datos a copiar.

En el caso de las copias de mysql no hay la posibilidad de copias incrementales con lo cual se generara una copia completa cada vez.
