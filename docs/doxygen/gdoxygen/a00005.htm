Estructura del directorio

|-- eventos

|   |   `-- 20-subversion.sh

|       `-- 10-subversion.sh
|-- extension.sh
|-- magcontrolversion
`-- menu.sh

Eventos

Al ejecutarse un evento se buscara dentro de la carpeta de cada plugin si contiene un script para ser ejecutado se tendrá en cuenta el orden según el nombre del script

Extensiones para los plugins

El archivo extension.sh es una porción de código que sera incluido dentro de la aplicación al arrancar con source.

Menú para los plugins

El archivo menu.sh si se encuentra se mostrará en la lista de componentes para ser incluido con source al ser seleccionada.
